# Sampling Variation Simulator

Single-page React + TypeScript app for visualizing sampling variation in the sample mean.

The app keeps the population mean fixed at `mu = 1.4` while allowing the population variance to change. It repeatedly draws samples, displays the current sample, and accumulates the sampling distribution of the sample mean.

## Public URL

After GitHub Pages deployment:

```text
https://jarebov.github.io/sampling-variation/
```

## How to run locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local URL shown by Vite, usually `http://localhost:5173/`.

## Build and preview

```bash
npm run build
npm run preview
```
